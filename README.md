DevLens is a sophisticated, full-stack SaaS tool designed to provide developers with a **"Hiring Mirror."** It performs industrial-grade audits on GitHub repositories to surface the technical signals and professional "vibe checks" that recruiters actually look for.

---

## 🌟 Key Features

### 🧠 Algorithmic Scoring Engine (V2)
DevLens has moved beyond "simple AI summaries." Our **Weighted Rubric system** enforces a multi-step audit process:
- **Baseline Scoring**: Every project starts at a base of 5.0.
- **Deductions**: Automatic penalties for missing Licenses (-1.0), missing Setup Guides (-1.0), and lack of Visual Demos (-0.5).
- **Merit Bonuses**: Extra points awarded for complex architecture (+1.5) and innovative application domains (+2.0).
- **The 7.0 "Ceiling"**: No matter how good the code is, if **Automated Tests** and **CI/CD** are missing, the score is hard-capped at 7.0.

### 💎 Premium UI/UX
- **Interactive Scoring**: A high-impact count-up animation for the final grade.
- **Glassmorphic Design**: A modern, sleek aesthetic with subtle background glows and film grain textures.
- **Framer Motion Transitions**: Fluid page transitions and staggered layout entrances.
- **Save Analysis**: Download audit results as a portable JSON file for your records.

### 🔬 Transparency & Debugging
- **Logic Scratchpad**: Every audit includes a raw mathematical breakdown of the AI's reasoning, making the scoring process clear and reproducible.
- **DETERMINISTIC ANALYSIS**: Set to `temperature=0.0` to ensure that identical repositories always receive identical scores.

## 🚀 Getting Started

### 1. Prerequisites
- **Groq API Key**: [Get it here](https://console.groq.com/keys)
- **GitHub Token**: [Generate a classic token](https://github.com/settings/tokens) (no special scopes needed for public audits)

### 2. Backend Setup
```bash
cd backend
python -m venv venv
# Windows
.\venv\Scripts\activate
# Mac/Linux
source venv/bin/activate
pip install -r requirements.txt
```
Create a `.env` in `backend/`:
```env
GROQ_API_KEY=your_key_here
GITHUB_TOKEN=your_token_here
```
Start the server:
```bash
uvicorn app.main:app --reload
```

### 3. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
Open [http://localhost:5173](http://localhost:5173).

## 🚢 Deployment to Vercel

DevLens is optimized for a single-click monorepo deployment.

1.  **Select the Root Folder**: Choose the top-level `DevLens/` directory in Vercel.
2.  **Environment Variables**: Add your `GROQ_API_KEY` and `GITHUB_TOKEN` in the Vercel Dashboard.
3.  **Automatic Routing**: The `vercel.json` file handles routing for both the React static files and the Python backend functions.
