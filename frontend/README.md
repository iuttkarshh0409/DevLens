# DevLens Frontend

Premium UI for the GitHub repository auditor.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
   (Requires `lucide-react`)

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Configure Backend URL:
   Update `API_BASE_URL` in `src/App.jsx` if necessary.

## Requirements

- Tailwind CSS
- Lucide React (icons)
